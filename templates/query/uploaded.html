Query - Upload Interactor Files

If you wish to query PaIntDB for interactions involving your interactors of interest, upload a txt, csv, or tsv file containing your interactors by clicking below (metabolites and protein interactors must be uploaded in separate files). If you wish to simply browse PaIntDB, skip this step.

File specifications:
  • Protein/gene and metabolite interactors should be uploaded in separate files.
  • For proteins/genes, use locus tag or UniProtKB ID.
  • For metabolites, use KEGG, ChEBI, EcoCyc, PubChem, or CAS ID to identify interactors. (Note that not all metabolites in the database have each of these IDs, so you should use ID types in the order of KEGG, ChEBI, EcoCyc, PubChem, CAS. Querying with more widely used IDs will increase the chance that your metabolites will be found in the database).
    • For metabolites, please select the type of ID being used before continuing
